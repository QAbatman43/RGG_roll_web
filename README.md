# RGG Roll Web

Веб-версия генератора Retro Game Gauntlet Roll Machine, сделанная в Godot 4.

Оригинальная программа была доступна только как `.exe`, поэтому эта версия пересобирает интерфейс и логику ролла для запуска в браузере.

Запустить: https://qabatman43.github.io/RGG_roll_web/

## Что умеет

- Выбирает случайный пункт из выбранного списка.
- Поддерживает списки платформ, спецролов, лутбоксов, "Кот в мешке", "Гитман" и "Колесо".
- Повторяет логику кнопок `Roll`, `STOP` и `На ультратоненького`.
- Поддерживает фильтр по первой букве или цифре.
- Показывает ссылки на YouTube, MobyGames, GameFAQs и RGG для выпавшего пункта, где это применимо.
- Для списка "Колесо" открывает описание пункта из `wheel_list.dat`.
- Может зачитывать выпавший пункт через TTS.
- Отдельно отключает звуки ролла кнопкой `MUTE`, не выключая TTS.

## Управление

- `Roll` запускает прокрутку.
- `STOP` начинает остановку ролла.
- `На ультратоненького` резко останавливает ролл после разрешённой задержки.
- Поле справа от `Roll` фильтрует список по первой букве.
- Чекбокс `Цифры` разрешает ввод цифр в поле фильтра.
- `TTS/OFF` включает или выключает озвучку.
- `MUTE/UNMUTE` включает или выключает только звуки ролла.
- Правый клик по кнопке списка открывает окно предпросмотра всего списка.

## Списки

Списки лежат в папке `lists`.

Для веб-версии списки хранятся как внешние файлы рядом с билдом, поэтому их можно обновлять в репозитории без нового экспорта Godot.

Важно для GitHub Pages:

- Файлы в `RGG_roll_web/lists` должны быть в UTF-8.
- Исходные списки оригинального приложения могут быть в UTF-16LE, но перед публикацией в web-папку их нужно перекодировать в UTF-8.
- После изменения только списков достаточно сделать обычный commit/push. Пересобирать Godot не нужно.


## Структура репозитория

- `index.html` - стартовая страница веб-версии.
- `rggweb.*` - файлы Godot Web export.
- `lists/` - внешние списки для ролла.
- `.nojekyll` - отключает обработку Jekyll на GitHub Pages.
